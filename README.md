# DAB Pumps for Homey

Control and monitor your DAB H2D water pumps directly from Homey.

## Features

- Monitor pressure, flow, power and temperature
- Detect pump activity and water flow
- Receive alerts when the pump goes offline or reports an alarm
- Control PowerShower mode
- Enable Sleep mode
- Use Homey Flow to automate your DAB pump

## Supported devices

- DAB Esybox (tested)
- Other DAB H2D-compatible pumps (limited support)

## Support

Open a GitHub Issue and include a diagnostic report.

## Notes

This is an unofficial integration and is not affiliated with DAB.
